	Comenzi din git des utilizate

rm -R filename		--> sterge fisierul (chiar daca e director)
rm -Rf filename 	--> sterge fisierul chiar daca stergerea produce erori

git log --pretty=format:"%h : %ar : %an : %s" --> format dragut pt. log

Config your user settings:
git config --global user.name "Ostoia Ovidiu"
git config --global user.email ostoia.ovidiu@gmail.com

git add *.java		--> add to the staging area only .java files
git add . 		--> add to the staging area all new files
git add filename	--> add to the stagin area only this filename

git commit -m "A commit message here"	--> commit files with a commit message
git commit -a -m "message"		--> commit all files even the files are not in the staging area yet

git status		--> show the status of your repo
